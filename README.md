# 📘 Miniguia de Estudos — PostgreSQL no NotebookLM

## 📌 Contexto e Objetivos

Este repositório apresenta um **miniguia de estudos básico sobre PostgreSQL**

O objetivo principal é apresentar os **fundamentos do PostgreSQL**, compreender seus principais conceitos e comandos SQL, além de praticar **engenharia de prompts** aplicada ao aprendizado técnico com IA.

---

## 📚 Curadoria de Fontes

Fontes abertas utilizadas como base de conhecimento:

- Documentação oficial do Postgres;
- Curso de PostgreSQL do canal Bóson Treinamentos do Youtube.

---

## 🧠 Engenharia de Prompts

Durante o estudo, foram elaborados e refinados prompts estratégicos para:
- Obter explicações didáticas e técnicas;
- Gerar exemplos práticos em SQL;
- Criar resumos estruturados e glossários.

📌 Aprendizado principal: **prompts claros e contextualizados geram respostas mais precisas e úteis**.

---

## 📖 Miniguia de Estudo

### Conteúdos Consolidados
- Fundamentos do PostgreSQL;
- Principais comandos SQL (DDL e DML);
- Estrutura de bancos, schemas e tabelas;
- Constraints e boas práticas.

### Glossário (Exemplos)
- **SGBD**: Sistema Gerenciador de Banco de Dados  
- **Schema**: Organização lógica de tabelas  
- **Primary Key**: Identificador único  
- **Foreign Key**: Relacionamento entre tabelas  

### Prompts Reutilizáveis
- *“Explique os principais conceitos do PostgreSQL com exemplos.”*
- *“Crie um resumo rápido sobre DDL e DML.”*
- *“Gere exercícios práticos de SQL no PostgreSQL.”*
